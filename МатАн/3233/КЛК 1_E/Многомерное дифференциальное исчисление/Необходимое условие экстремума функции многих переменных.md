$f : D \subset R^m \to R, \quad x_0 - \text{внутренняя точка } D$

Если ты $x_{0}$ - точка экстремума $f$, то все частные производные равны нулю или не существуют. (то есть $\text{grad}f = 0_{m}$)
$$\frac{\delta f}{\delta x_{i}} (\forall i = 1,\ldots,m) = 0 \text{ или } \not \exists$$
## Док-во
$$x_{0} = (x_{1_{0}},x_{2_{0}},\ldots,x_{m_{0}}), \quad x = (x_{1},x_{2},\ldots,x_{m})$$
$$ x_{1} = t, x_{2} = x_{2_{0}},\ldots,x_{m} = x_{m_{0}}$$
$$\phi(t) = f(x)$$
$$t = x_{1_{0}} - \text{точкак эктремума функции } \phi(t) \implies \phi'(x_{1_{0}}) = 0 \text{ или } \not \exists \implies f'_{x_{1}}(x_{0}) = 0 \text{ или } \not \exists$$
