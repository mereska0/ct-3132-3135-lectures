### Указатели↗️

```
#include <stddef.h>
int main()
{
	int* ptr;//указатель на тип инт
	char * ptr2;
	float *ptr3;
	//можно писать всеми тремя вариантами
	
	int *a, b; // а - указатель, а b - просто инт
	int *c, *d; // тут уже оба указатели
	
	printf("%zu\n", sizeof(int*));
}
``` 

`stddef.h` нужен  ==для определения размеровсмещений и указателейобеспечивая переносимость кода между платформами==. Он необходим для использования `NULL``size_t` (результат `sizeof`)`ptrdiff_t` (разница указателей) и `offsetof`.

Базово указатель весит 8 байт, но это не всегда так, это зависит от битности системы(в нашем случае базово 64 бита)
НО: добавив аргумент в компилятор `-m32` мы получим что указатель весит уже 4 байта

Рекомендуется сразу инициализировать сразу, чтобы там не хранился мусор:
```
int *ptr = NULL;

char c = '\n';
char *ptr2 = &c;//указатель на переменную с

float arr[5] = {1, 2, 3, 4, 5}
float *ptr3 = arr;
```
Указатель на массив = указатель на первый элемент массива
По сути своей массив это тоже указатель на его первый элемент

Арифметика указателей работает за счет того, что указатели хранятся в памяти друг за другом:
```
float arr[5] = {1, 2, 3, 4, 5}
float *ptr3 = arr;

ptr3++;//изменяется не на 1 а на 1*sizeof(float)(смещается в следующую ячейку)
printf("%g\n", ptr3)//выведет второй элемент

ptrdiff_t aa = ptr - ptr3;//хранит разность указателей(вряд ли понадобится)

float neptr = *ptr3//разыменовали поинтер
```
можно из поинетра вычитать поинтер и получить их дельту, но нужно делать это только если оба поинетра указывают **на одну ячейку**

фактически есть `nullptr`, но это завуалированный `NULL`

Когда мы передаем функции переменную мы создаем **копию**, и не трогаем внешнее значение переменной, но передав *указатель* мы можем влиять на нее

 ```

 }
 ```
Результатом кода выше будет error  тк мы будем пытаться обращаться к локальной переменной которой не будет существовать

&arr;//все еще указатель на первый элемент массива

const могут быть и указатели(очев)
```
const int* p;//указатель на константу типа int
int *const q = &x;//константный указатель
```
по стандарту нельзя не инициализировать константный указатель(ну типа чзх создавать пустую константу)
