## Конструкция выбора(switch case)
 только целочисленные значения, плавающие точки плывут нахуй 

```
int x;
switch(x)
{
	case 3: x = x*2;
	case 1: x = x-1;
	
	case 4:
	case 5: x = x-2 \\ для кейса 4 и 5 
	
	default: x = x/2 \\ если поставить в начале, то будет выпольнятся и выход
}
return 0;

```

По сути Swith это goto на нужный case, если нет подходящего case то просто выход из конструкции


### Оптимизация через разворот цикла

разворот цикла частая оптимизация, которую сейчас даже можно через компилятор настроить
#### Обычный разворот цикла
```
int number = 12; 
int count = number / 4;
int i = 0;
while (number-- > 0) \\loop unrolling
{
	++i; ++i; ++i; ++i;

}
printf("%i\n", i);

```

#### через switch case
```
int number = 12; \\ min = 1
int count = number / 4;
int i = 0;
switch( number % 4)
{
case 0: 
	do
	{
	++i;
	case 3: ++i;
	case 2: ++i;
	case 1: ++i;
	} while (--count > 0)
}
printf("%i\n", i);
```

меньше инструкций, что кайф было лет 100 назад, сейчас как будто похуй. Если итераций много то это эффективно.

## Псевдоним
Псевдонимы в C++ позволяют создавать альтернативные имена для существующих типов данных. Это улучшает читаемость кода, упрощает поддержку и делает код более гибким.

```
typedef unsigned Sasha_privet; // называешь этот тип переменных своим названием 

int main() {
	int z = 2;
	int x = z;
	
	Sasha_privet c; // можно теперь называть unsigned = Sasha_privet
	
	return 0;
}

```
