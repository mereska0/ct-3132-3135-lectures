### Побитовые операторы
**Битовый сдвиг:**
- арифметический - дублирование числа
- логический - добавка нулей
```
int main()
{
int8_t a = 5;
int8_t b = a << 2; //логический сдвиг
uint8_t c = a >> 1; //арифметический сдвиг
printf("%i = %b << %i\n", b, a, 2); //20 = 101 << 2

return 0;
}
```
Если сдвиг слишком большой, то получаем 0

При сдвиге отрицательного числа ничего хорошего не будет

В положительных числах сдвиг эквивалентен делению/умножению на $2^n$ (и это мега эффективней обычного деления/умножения)

***Инверсия битов:*
- ~8;
***Побитовое и/или:*
- 8 & 14;
-  6 | 14;
**Логическое отрицание:**
-  !b;
**XOR:**
-  a ^ b;

### Плавающая точка
```
float f = 0.1f; //IEEE-754 single precision
//если не написать f то просто скастится в double
double d = 0.0;//IEEE-754 double precision
long double ld = 0.0;//чудище

printf("%lf\n", d);
printf("%f\n", f);
```
- float: от $-10^{38}$ до $10^{38}$
- double:  от $-10^{308}$ до $10^{308}$

```
printf("%g", 123.456);  // Выведет: 123.456
printf("%g", 123.456789); // Выведет: 123.457 (по умолчанию 6 значащих цифр)
printf("%.3g", 123.456789); // Выведет: 123 (3 значащие цифры)
printf("%g", 0.0000123); // Выведет: 1.23e-05
```

Можно писать как 0.5 так и .5, как 1.0 так и 1.
В числах с плавающей точкой операции **не ассоциативны**(пиздец)
Также результат зависит от компилятора😭😭😭
```
float a = 1.f / 0.f;
// float 1.f - 4 байта
// double 1.0 - 8 байт
float i = INFINITY;
printf("%f\n", a);//inf
```
inf это float, и достигнуть его возможно именно делением float'ов
Если попытаться вывести 1/0, то clang кинет warning, но все равно откомпилит и выведет просто мусорное значение. gcc выводит код ошибки

float NAN(Not a Number) - математическая неопределенность(0/0, inf/inf и т.д.)
nan разные компиляторы выводят по-разному

nan == nan   - false (прикол)
также есть функция чтобы узнать nan ли число, но способ выше намного быстрее

Nan бывает:
- сигнальный(аппаратное исключение)
- тихий (появится и спокойно будет жить весь цикл жизни программы)

```
bool a1 = 0.5;// скастится 1
int8_t a2 = 0.5;//скастится 0
```
