### Variadic functions

это функции с переменным числом аргументов
например функция `printf(...)` которая работает от n аргументов
```
int add_nums(size_t n, ...)
{
	int res = 0;
	va_list args;
	va_start(args, n);
	for(size_t i = 0; i < n; i++)
	{
		res+= va_args(args, int);
	}
	va_end(args);
	return res;
}
```
