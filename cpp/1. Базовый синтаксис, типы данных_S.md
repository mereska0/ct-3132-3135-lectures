код пишется в функциях, главная функция `int main(void);`(да ну нахуй)
```
#include <studio.h> стандартная библиотека

int main(void)
{
int a, b;
scanf("%i %i", &a, &b); "%i %i" - формат ввода, &a, &b ссылка на переменые
printf("a + b = %i\n", a + b); стандартный вывод, %i - формат вывода
printf("a + b = %#x\n", a + b);  
return 0
}
```
*scanf возвращает int, значит по результату можно понять что считалось и считалось ли что-то вообще*

### Типы данных:
- int (до $2^{32}$)
- short(до $2^{16}$)
- long(зависит от компилятора $2^{32}$ или $2^{64}$) %l
- long long(до $2^{64}$)9
- char(до $2^8$)
-  \_Bool или bool(нужен `<stdbool.h>` или не надо блять я так и не понял)
- \_BitInt(n) выделяет n битов (не больше 8 по идее)

также можно самому задавать " фиксированную ширину":
int8_t, int16_t, int32_t, int64_t  *PS: это круто, это классно, это свэг*

**long и int128_t не свэг!!**

size_t - тип данных который хранит битность программы(классически 64 бита, редко 32) %zu

чтобы хранить беззнаковые переменные перед обьявлением пишется unsigned:
unsigned int a;
Базово все переменные signed, кроме `char`. Для него нужно отдельно написать
___
### Спецификаторы:
- %i - универсальный
- %d - 10-ичные числа
- %x - 16-ричные числа
- %o - 8-ричные числа
- %b - 2-ичные числа
- %lli - long long
- %h - short
- %l - long
- %zu - size_t
- %з - pointer(начнется в [[4. Указатели_S]])
