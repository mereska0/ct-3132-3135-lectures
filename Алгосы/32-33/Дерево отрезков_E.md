## Построение
$O(n)$
```
def build(x, lx, rx, a) 
	// если мы лист
	if (rx - lx <= 1)
		tree[x] = a[lx] // аккуратно если мы до степени двойки достраивали
		return
	int mid = (lx+rx)/2
	build(2x+1, lx,mid)
	build(2x+2,mid,rx)
	tree[x] = tree[2x+x] + [2x+2] //зависит что мы строем
```

## Обновляем значение в дереве отрезков на сумму 
$O(\log n)$
```
a = []
n - size(a)

int size = 1
while (size<=n) { // доводим размер массива до степень двойки 
size*=2
}
tree.build(2*size-1) // дерево отвечающая за сумму на полуинтервале 

// те значение которые мы добавили до степени двойки мы заполняем нулями (нейтральными значениями для операции, в нашем случае сумма)
----------------------------------------------------------------------

def set_value(index, val, x, lx, rx)
// index - индекс меняемого элемента, val - значение которое меняем, x - индекс в дереве, lx, rx - границы полуинтервала (lx,rx)
	if (rx - lx <= 1) \\если попали в лист(самый низ)
		tree(x) = value
		return
	int mid = (lx + rx)/2 \\ считает середину полуинтервала
	
	// пытаемся понять в какой половине полуинтервала сидит индекс
	if (index < mid) 
		// переходим в левый лист и сдвигаем границы полуинтервала 
		set_value(index, val, 2x+1, lx, mid) 
	else 
		// переходим в правый лист и сдвигаем границы полуинтервала
		set_value(index, val, 2x+2, mid, rx)
		
	// обновляем дерево отрезков с учетом нового значения
	tree[x] = tree[2x+1] + tree[2x+2]


```

## Получение суммы элементов
$O(\log n)$
```
// получаем сумму на полуинтервале
def get_sum(l,r,x,lx,rx)
// l,r - границы полуинтервала, x - индекс в дереве, lx,rx - границы локального полуинтервала
// полностью не лежит 
if (r <= lx || rx <= l) 
		return 0; // нейтральй элемент для операции
// полностью лежит
if (l <= lx && rx <= r)
	return tree[x]
// частично лежит
int mid = (lx + rx) / 2
return get_sum(l,r,2x+1,lx,mid) + get_sum(l,r,2x+2,mid,rx)
```
